# Get-WinSCPItemChecksum

## SYNOPSIS
Calculates a checksum of a remote file.

## SYNTAX

### Set 1
```
Get-WinSCPItemChecksum [-WinSCPSession] <Session> [-Algorithm] <String> [-Path] <String[]> [<CommonParameters>]
```

## DESCRIPTION
